# js-decoder

## Running Locally

Make sure you have [Node.js](http://nodejs.org/) installed.

```sh
cd js-decoder
```

```sh
npm install
```

```sh
npm start # or node app.js 
```

Your app should now be running on [localhost:5000](http://localhost:5000/) or [ipv4address:5000](http://192.168.X.X:5000/)
